# useUseless

A completely useless React hook that does absolutely nothing.

## Installation

You can install the package from npm:

```bash\
npm install @prajuwalsingh/use-useless

Usage

Import useUseless from @prajuwalsingh/use-useless and use it in your functional component:

import React from 'react';\
import { useUseless } from '@prajuwalsingh/use-useless';

function MyComponent() {\
  useUseless(); // This does absolutely nothing\
  return <div>Hello, World!</div>;\
}

export default MyComponent;


Why?

Because sometimes, doing nothing is the most useful thing you can do.\
